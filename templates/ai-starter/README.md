# ai-ml-starter

Starter project generated by `ai-ml-dev-bootstrap`.

## Commands

```bash
uv run python scripts/check_env.py
uv run jupyter lab
uv run ruff check .
uv run pytest
```

## Profiling example

```bash
uv run python scripts/profile_torch.py
uv run tensorboard --logdir runs
```

## Dependency files

- `requirements/base.txt`: common scientific Python stack.
- `requirements/llm.txt`: Hugging Face / LLM tooling.
- `requirements/mlsys.txt`: profiling, benchmarking, export/runtime tooling.
- `requirements/personal.txt`: external-service-friendly personal tooling.
- `requirements/enterprise.txt`: local/compliance-oriented tooling.
- `requirements/cuda-extras-linux.txt`: experimental CUDA extras, Linux/WSL only.
